## Team Members
- Nicolas Tran
- Diego Mejia

## How to Run
1. Clone the repository:
   git clone https://github.com/chablades/CS3560-Prog2-Graphics_Implementation

2. Open the project in IntelliJ

3. Run the main class:
   src/main/java/Main.java

## Features Implemented
- Random secret word from dictionary
- 6-turn guessing system
- Color-coded feedback (green, yellow, gray)
- On-screen keyboard with letter status
- Save/load game state
- Duplicate-letter handling (true Wordle logic)
- Full MVC architecture

## Controls
- Type letters on your keyboard to enter a guess
- Backspace: delete last letter
- Enter: submit guess
- File → Save Game
- File → Load Game
- File → New Game
